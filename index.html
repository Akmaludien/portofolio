import tensorflow as tf
from base_predictor import BasePredictor

# Modular OOP Bi-LSTM for AWS/AAWS/ARG
class AWSPredictor(BasePredictor):
    def __init__(self, station_id):
        super().__init__(station_id, n_features=17)
        self.model = self.build_bilstm()

    def build_bilstm(self):
        return tf.keras.Sequential([
            tf.keras.layers.Bidirectional(
                tf.keras.layers.LSTM(128, return_sequences=True)),
            tf.keras.layers.Dropout(0.2),
            tf.keras.layers.Dense(1)
        ])

Research

Undergraduate Thesis

Sistem Monitoring & Prediksi Curah Hujan Jawa Barat

Developed specifically for STMKG Final Year Project. An intelligent real-time IoT weather monitoring platform integrated with a Deep Learning (Bi-LSTM) prediction engine. The AI uses a Modular OOP approach with dynamic feature engineering (AWS 17 features, AAWS 10, ARG 1) and an Adaptive Seasonal Filter to suppress over-prediction during the dry season. Built with Node.js, Python, InfluxDB, and deployed via Docker on Coolify.

Deep Learning Bi-LSTM IoT Telemetry Time Series Meteorology
Published Paper

Air Quality Prediction System Using Telegram Bot Based on Real-Time Data

Published in Journal of Computation Physics and Earth Science (JoCPES), Vol. 3, No. 1 (2023). This research focuses on developing a system that utilizes a Telegram bot to provide real-time air quality information and 5-day forecasts to improve public access to environmental data in Indonesia. The system integrates real-time data using the OpenWeatherMap API and employs a Linear Regression-based machine learning model to generate air quality projections.
