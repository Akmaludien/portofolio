# Akmaludien Ramadhan — Portfolio

A premium, single-page personal portfolio website showcasing my skills, projects, and research in Instrumentation Engineering, Machine Learning, and Climate Data Analytics.

## 🌐 Live Preview

[Visit Portfolio](https://akmaludien.github.io/portofolio)

## ✨ Features

- **Premium Design** — Modern, minimal aesthetic inspired by Vercel/Linear/Apple
- **Dark Mode** — Elegant dark theme with cyan accent
- **Responsive** — Fully responsive across all devices
- **Smooth Animations** — Scroll reveals, typing animation, animated counters, particle background
- **SEO Optimized** — Meta tags, Open Graph, Twitter Cards, structured data (JSON-LD)
- **Accessible** — ARIA labels, semantic HTML, keyboard navigable
- **Lightweight** — Pure HTML/CSS/JS, no frameworks, fast loading
- **Glassmorphism** — Modern glass-effect cards and navigation

## 🛠 Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties, Animations, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, JetBrains Mono)

## 🚀 Deploy

This site is designed for GitHub Pages:

1. Go to **Settings > Pages**
2. Set source to **Deploy from a branch**
3. Select **main** branch, root folder
4. Save — your site will be live at `https://akmaludien.github.io/portofolio`

## 📝 Customization

Edit the content directly in `index.html`. Key areas to update:
- Contact information (email, GitHub, LinkedIn, Instagram)
- Profile photo
- CV/Resume PDF
- Project links and descriptions
- Experience timeline
- Certificates
